# E-Commerce Sales Analysis Dashboard

## Project Overview
This project analyzes **e-commerce sales performance using Excel and Power BI** to uncover revenue trends, profitability insights, and regional sales patterns.  
The interactive dashboard enables **KPI tracking, category performance evaluation, and geographic sales insights** to support data-driven business decisions.

---

## Tools & Technologies
- Excel  
- Power BI  
- Data Cleaning  
- Data Visualization  
- Exploratory Data Analysis (EDA)

---

## Key Metrics Analyzed
- **Revenue:** $2.29M  
- **Profit:** $286K  
- **Orders:** 9,994  
- **Units Sold:** 37,873  
- **Profit Margin:** 12.47%

---

## Key Insights
- Identified **20–28% YoY growth trends** across revenue, orders, and quantity.  
- Discovered **high-profit product categories** driving business performance.  
- Analyzed **regional sales distribution** to highlight top-performing states.  
- Evaluated **top 10 sub-categories** contributing to overall sales.

---

## Dashboard Features
- KPI cards for **Revenue, Profit, Orders, and Growth**
- **Category-wise profit analysis**
- **Sales and profit trend visualization**
- **Geographic sales distribution by state**
- **Top-performing product sub-categories**
- Interactive filters for **Year, Segment, and Region**

---

## Business Problem Solved
This dashboard helps stakeholders **monitor sales performance, identify profitable categories, and evaluate regional demand**, enabling **data-driven product and revenue strategy decisions**.
